# AI Resume Screening System

A full-stack NLP project that analyzes resumes against job descriptions using Sentence-BERT embeddings, cosine similarity, skill extraction, and ATS-style scoring.

## Features

- Upload resume PDF
- Paste job description
- Extract text from PDF
- Semantic similarity scoring
- ATS-style scoring
- Skill extraction
- Matched skills and missing skills
- Resume improvement suggestions

## Tech Stack

### Backend
- Python
- FastAPI
- Sentence Transformers
- scikit-learn
- PyMuPDF

### Frontend
- React
- Vite
- Axios
- CSS
